# KPI Bếp - Quản lý tồn & kế hoạch chuẩn bị ca sáng

Web app tĩnh giúp bộ phận bếp:

- Nhập tồn cuối ngày (đậu, bò, nước dùng).
- Nhập số bát dự kiến bán trong giờ cao điểm ngày hôm sau.
- Tự động tính:
  - Khối lượng nguyên liệu cần chuẩn bị thêm.
  - Số mẻ cần thực hiện theo năng lực bếp.
  - Tổng thời gian chế biến.
  - Mốc giờ cần hoàn thành nếu bắt đầu từ một thời điểm cụ thể (ví dụ 06:00).

## Chạy local

Mở trực tiếp `index.html` hoặc chạy server tĩnh:

```bash
python3 -m http.server 8000
```

Sau đó truy cập: <http://localhost:8000>
